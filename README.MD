# mini-network-monitor-ssd1306

A simple home-made network monitor working on a 128x64 OLED display(SSD1306 I2C) with raspberry pi.

## Demo

[Video demo](https://www.youtube.com/watch?v=upcZGkevB1Y)

## Supported devices(OS)

- UBNT EdgeOS
- Generic Device(ICMP)
